# TrigaServer

TrigaServer is a software for GNU operating system to get the real-time values of the Nuclear Reator Triga IPR-R1 and share in network.

## Clone and compiling

``` Bash
git clone https://github.com/CDTNuclear/TrigaServer.git
mkdir -p build
cd build
cmake ..
make
```
